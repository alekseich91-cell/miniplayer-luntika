# Devlog — Miniплеер Лунтика

## 2026-04-13: v1.0.0 — Первый релиз

### Что сделано
- Полный плеер с нуля за одну сессию
- Фоновая музыка: плейлист по кругу / один трек по кругу, Play/Pause/Stop
- Джинглы: кнопка ▶ на каждом, индивидуальная громкость, табы-папки для организации
- Кроссфейд: настраиваемый fade out/in (0-5 сек), джингл стартует мгновенно
- Drag & drop MP3/WAV файлов
- Сохранение состояния между запусками (JSON)
- Сборка: macOS .app + .dmg с ad-hoc подписью, Windows .exe через GitHub Actions

### Архитектурные решения
- **pygame.mixer.music** для фона вместо Channel — лучшая поддержка MP3 стриминга
- **QTimer fade** вместо pygame.fadeout() — fadeout() останавливает канал, теряя позицию
- **get_busy() polling** вместо pygame.event — event требует pygame.display.init(), который создаёт лишнее окно
- **Прерывание вместо очереди** для джинглов — пользователю удобнее мгновенная реакция

### Баги пойманные и исправленные
1. **SIGABRT crash** при запуске — pygame.event.get() без display init. Заменено на get_busy() polling.
2. **Джинглы не триггерились** — mouseDoubleClickEvent не доходил до JingleRow из-за child widgets. Заменено на кнопку ▶.
3. **Джинглы играли все подряд** — очередь копила клики. Убрана очередь, новый джингл прерывает текущий.
4. **macOS "повреждено"** при скачивании — добавлена ad-hoc подпись и DMG.

### Стек
Python 3.13 + PyQt6 + pygame.mixer + PyInstaller
